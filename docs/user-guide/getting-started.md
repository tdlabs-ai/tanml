# Getting Started

This guide will walk you through your first validation run with TanML.

## 1. Launch the Dashboard

```bash
tanml ui
```

## 2. Load Data

TanML supports CSV files. Upload your training and testing datasets in the "Data Ingestion" tab.

## 3. Preprocessing

Use the "Preprocessing" tab to handle missing values and encode categorical variables.

## 4. Model Training & Validation

Navigate to "Model Evaluation" to train a benchmark model (XGBoost) and view validation metrics (ROC, PRC, Matrix).

## 5. Generate Report

Click "Generate Report" to download a full `.docx` audit report.
